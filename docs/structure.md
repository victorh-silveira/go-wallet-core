**Estrutura do Projeto**

Este projeto segue organizacao em camadas para Go:

- `src/application`
- `src/domain`
- `src/infrastructure` (repositorios em memoria em `repository/memory`)
- `src/interfaces`
- `tests/unit` (`domain/entity`, `application`, `handler`, `infrastructure`), `tests/integration`, `tests/e2e`, `tests/contract`, `tests/support`
